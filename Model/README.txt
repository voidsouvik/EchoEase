class_dict:
The CSV file class_dict is the area where the classes are stored along with the indexes. 
This file will give the output of the model.

efficientnet.ipynb:
This file is the file for the Jupyter Notebook of the model. 

EfficientNetB3-ASL-99.64.h5:
The keras model that is stored as a in the HDF5 format for use in apps and programs.

converted_model.tflite:
The keras model, converted to tflite format, for use for Android Studio.

handsign.py:
The sample application of the program, where the model is used and a sample image is added 
as input and tested.
